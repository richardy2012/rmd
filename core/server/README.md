
#### API

- GET `/api/v1/user/me`
- GET `/api/v1/post?offset=0&limit=15`
- GET `/api/v1/post/:id`
- POST `/api/v1/post`
- PUT `/api/v1/post/:id`
- DELETE `/api/v1/post/:id`

- GET `/api/v1/media?offset=0&limit=15`
- GET `/api/v1/media/:id`
- DELETE `/api/v1/media/:id`

- POST `/api/v1/authenticate/token`
- POST `/api/v1/authenticate/revoke`

- POST `/api/v1/upload`
- POST `/api/v1/upload/base64`
